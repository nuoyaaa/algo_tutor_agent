# Algorithm Tutor Agent

一个面向数据结构与算法学习场景的 RAG 系统。  
项目目标是帮助用户从分散的算法学习资料中检索高质量内容，并基于检索结果生成更符合“定义 / 例子 / 区别”等学习意图的回答。

---

## 1. 项目背景

在学习数据结构与算法时，高质量资料通常分散在不同平台与文章中。  
同一知识点下，用户的问题也不只是“是什么”，还可能是：

- 定义是什么
- 能举个例子吗
- 和另一个算法有什么区别
- 适合解决什么问题

因此，本项目尝试构建一个面向算法学习场景的 RAG 系统，实现：

- 本地知识库构建
- 语义检索
- 检索结果去重
- 基于学习意图的 rerank 精排
- 基于检索上下文的答案生成
- 来源回显与分数展示

---

## 2. 当前功能

### 已实现
- 基于 Markdown 文档构建本地知识库
- 使用 sentence-transformers 生成 embedding
- 使用 FAISS 建立向量索引
- 语义召回候选 chunk
- 候选语义去重
- source 多样性控制（同一来源最多保留一条）
- 规则化 rerank（区分主题词与意图词）
- 使用 OpenAI 模型基于检索结果生成回答
- 输出来源、retrieval score 与 rerank score

---

## 3. 系统架构

```text
用户输入 query
→ query embedding
→ FAISS 召回候选
→ 语义去重
→ rerank 精排
→ source 多样性控制
→ 构造上下文
→ LLM 生成答案
→ 返回答案与来源
